# Walmart-Ecommerce-Sales-Data-Pipeline
This project develops a data pipeline to analyze Walmart's e-commerce sales trends around public holidays. It merges grocery sales data with complementary datasets, cleans and transforms the data, and calculates average weekly sales. The final outputs include cleaned datasets and analytical summaries for insights into consumer behavior.

## Key Features
- **Data Extraction**: Integrates grocery sales data from a PostgreSQL database with additional datasets.
- **Data Transformation**: Cleans the data by addressing missing values and extracting relevant metrics, such as monthly sales figures.
- **Analytical Insights**: Calculates average weekly sales and identifies key drivers affecting sales performance.
- **Output**: Provides cleaned datasets and analytical summaries in CSV format for integration into analytical workflows.

## Technologies Used
- Python
- Pandas
- PostgreSQL
